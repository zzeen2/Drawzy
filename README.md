#  LuckyDraw Token (LDT) 🎲

> **가격대별 랜덤뽑기로 상품을 획득하는 Web3 마켓플레이스**

## 📋 프로젝트 개요

### 🎯 핵심 컨셉
- 판매자가 **실제 가격**을 표시하며 상품 등록
- **가격대별 카테고리**로 자동 분류 (1 HBT, 2 HBT, 3 HBT)
- 구매자는 각 카테고리에서 **랜덤뽑기**로 상품 획득
- **공정한 기대값**: 투자한 토큰만큼의 가치 보장

### 🏆 목표
- **재미있는 쇼핑 경험**: 랜덤뽑기의 스릴
- **공정한 거래**: 가격대별 분류로 합리적 기대값
- **활발한 상품 등록**: 판매자들의 적극적 참여 유도

## 🎲 핵심 시스템

### 1. 🏷️ 가격대별 카테고리
- **1 HBT 뽑기**: 실제가치 1,000원 ~ 5,000원 상품
- **2 HBT 뽑기**: 실제가치 5,000원 ~ 10,000원 상품  
- **3 HBT 뽑기**: 실제가치 10,000원 ~ 20,000원 상품
- **5 HBT 뽑기**: 실제가치 20,000원 이상 프리미엄 상품

### 2. 🎁 상품 등록 시스템
- **판매자 등록**: 상품명, 설명, **실제 가격** 표시
- **자동 분류**: 실제 가격에 따라 카테고리 자동 배정
- **투명성**: 등록된 상품의 실제 가치 공개

### 3. 🎰 랜덤뽑기 메커니즘
- **카테고리 선택**: 원하는 가격대 카테고리 선택
- **토큰 지불**: 해당 카테고리의 토큰 지불
- **랜덤 추첨**: 해당 카테고리 내 상품 중 랜덤 획득

## 🎯 랜덤뽑기 시스템

### 🎲 뽑기 메커니즘
```solidity
function drawFromCategory(uint8 categoryLevel) external {
    require(balanceOf(msg.sender) >= categoryLevel, "토큰 부족");
    
    // 토큰 소각
    _burn(msg.sender, categoryLevel * 10**18);
    
    // 해당 카테고리에서 랜덤 선택
    uint256 randomIndex = _generateRandom() % categoryItems[categoryLevel].length;
    Product memory wonProduct = categoryItems[categoryLevel][randomIndex];
    
    // 사용자에게 상품 지급
    userProducts[msg.sender].push(wonProduct);
}
```

### 🔄 공정성 보장
- **동일 기대값**: 각 카테고리 내 상품들의 평균 가치 = 뽑기 비용
- **투명한 풀**: 각 카테고리별 등록된 상품 목록 공개
- **검증 가능**: 블록체인에 모든 거래 기록

### ⛓️ 스마트 컨트랙트
```solidity
contract HealthBoostToken is ERC20 {
    struct Product {
        string name;
        string description;
        uint256 realPrice;      // 실제 가격 (원)
        uint8 category;         // 카테고리 레벨 (1,2,3,5)
        uint8 imageIndex;       // 이미지 인덱스
        address seller;         // 판매자
    }
    
    // 카테고리별 상품 저장
    mapping(uint8 => Product[]) public categoryItems;
    
    // 사용자별 획득 상품
    mapping(address => Product[]) public userProducts;
}
```
